Responsive Web Design

What is Responsive Web Design

Responsive Web Design is a method of creating websites off of the idea of them being responsive to the user based on their behavior and environment they are accessing the website from. When creating a RWD it takes using layouts of some sorts to get the idea of how a website will unfold depending on how the user is viewing it. Within the code of website there are CSS media queries that help a lot to make the website flexible within the bounds of most platforms.
